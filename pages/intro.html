Introduction

  • This web software utilizes the computer's audio interface to provide a real-time oscilloscope, FFT spectrum analyzer, and waveform generator (sine, square, triangle).
  • The microphone input is used for acquiring analog signals, while the audio output allows for controlled emission of synthetic signals.
  • Ideal for electronic testing, acoustic analysis, and DSP experimentation via browser, without installation.
This software rely on the most modern browser capabilites, including:
  • Web Audio API
  • Media Source API
  • Media Capture and Streams API
  • Media Output Devices API
